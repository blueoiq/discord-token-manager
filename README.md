# discord-token-manager
a script for controlling a discord account from the users token

please dont use this maliciously and only on accounts you have consent of, obviously

  ## User info
- User info (email, phone, 2fa, ect)
- Relations
- Get direct message channels
- Get connections

  ## Guild management
  - list the users guilds
  - view guild information
  - leave a guild
  - get guild preview
  - get guild emojis
  - get guilds stickers
  - get guild template
  - get guild welcome screen
 
  ## Channel management
  - list channels in a guild
  - create text channel
  - create voice channel
  - create catergory
  - edit channel
  - delete channel
  - create invite
  - get channel messages
  - start typing indicator for a channel
 
    ## Messages
    - send a message
    - send an embed
    - delete message
    - get a channels messages
   
    ## Member management
    - get guild members
    - view users permissions
    - ban user
    - kick user
    - add a member to guild
    - get guild bans

      ## Role management
      - get guild roles
      - create role
      - add role to a member
      - remove role from a member

      ## Webhooks
      - create webhook
      - send message with webhook

     ## Security
    - get guild audit logs
    - get guild intergrations
    - get guild prune count
    - begin guild prune
    - get guild vanity url
    - get guild widget
    - get guild permissions

    - create guild template
